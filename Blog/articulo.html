¿Qué es el HTML?

HTML son las siglas de Hyper Text Markup Language (lenguaje de Marcado de hipertexto), es el lenguaje estándar utilizado para crear y estructurar páginas web. Fue desarrollado por Tim Berners Lee en 1991 y ha evolucionado a lo largo de los años.

HTML permite definir la estructura básica de una página web mediante el uso de etiquetas. Estas etiquetas inducan al navegador como debe mostrar el contenido, como texto, imágenes, videos y otros elementos multimedia.
El código HTML se compone de una serie de etiquetas que se escriben entre corchetes angulares (<>). Cada etiqueta tiene una función específica y puede contener atributos que proiporcionan información adicional sobre como debe mostrarse el contenido.
